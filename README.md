# CoursesApp1000
Это рабочая версия мобильного приложения для просмотра и управления курсами. Данный коммит содержит полностью реализованную навигацию, UI и основную бизнес-логику. Работа с базой данных (Room) будет реализована позже. Размытие секциий с информацией на карточке курса будет добавлено в отдельной ветке (медленный рендер).

**Стек технологий:**  
- Kotlin
- Retrofit
- Android Jetpack (ViewModel, LiveData, RecyclerView)  
- Coroutines & Flow  
- Dagger
- Navigation Component  
- AdapterDelegates для RecyclerView
- MVVM
- XML
- Clean Architecture
- Многомодульность

**Функциональность на текущий момент:**  
- Экран входа с валидацией полей  
- Главный экран с отображением списка курсов  
- Возможность отмечать курсы как избранные  
- Экран избранного с отображением только избранных курсов  
- Сортировка курсов по дате публикации   

**Планы на будущее:**  
- Интеграция с Room для локального хранения курсов
